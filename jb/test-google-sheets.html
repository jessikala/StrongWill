📋 How to Fix "Failed to fetch"

1 Open Google Apps Script

Go to your Google Sheet → Extensions → Apps Script

2 Click "Deploy" → "Manage deployments"
3 Click the Edit button (pencil icon)
4 Set "Who has access" to "Anyone"

⚠️ This is the most common issue! It must be "Anyone", not "Only myself" or "Anyone with Google Account"

5 Click "Deploy"
6 Authorize if prompted

Click "Authorize access" → Select your account → Advanced → "Go to [project name] (unsafe)" → Allow

7 Copy the Web App URL and paste it above

✅ What You Should See

When the URL is working correctly, you should see:

{"success": true, "bookings": [...]}

✅ If you see this, your Google Sheets integration is working!

❌ Common Errors

Error: "Failed to fetch"

→ Deployment access is not set to "Anyone"
→ Wrong URL or deployment doesn't exist
→ Deployment is archived/inactive

Error: "Authorization required" or "Access denied"

→ Set "Who has access" to "Anyone"
→ Re-authorize the script

Error: 404 Not Found

→ Wrong URL
→ Deployment was deleted
→ Copy the URL again from Apps Script
